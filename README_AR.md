# نظام إدارة الموظفين والحوافز

نظام متكامل لإدارة الموظفين مع نظام حوافز تلقائي ونظام تتبع العملاء والأجهزة.

## المميزات الرئيسية

### نظام الحوافز التلقائي

يقوم النظام بحساب وإضافة الحوافز تلقائياً في الحالات التالية:

- **حافز حضور العميل** (50 جنيه): يُضاف لموظف الكول سنتر عندما يحضر العميل للشركة
- **حافز الحضور في الموعد** (20 جنيه): يُضاف عند تسجيل الدخول في الموعد المحدد
- **حافز العمل في الإجازة** (100 جنيه): يُضاف عند العمل في يوم إجازة رسمية
- **حافز الساعات الإضافية** (30 جنيه/ساعة): يُضاف عند العمل ساعة أو ساعتين إضافية
- **حافز الهدف الشهري** (500 جنيه): يُضاف عند إكمال 9999 عميل بنسبة حضور 50%
- **حافز تسجيل العميل** (30 جنيه): يُضاف لموظف الريسبشن عند تسجيل حضور عميل

### نظام الأكواد الذكي

كل عميل يحصل على كود فريد يربطه بموظف الكول سنتر الذي استقبله، مما يسهل:
- تتبع أداء الموظفين
- حساب الحوافز تلقائياً
- متابعة العملاء

**مثال على الكود**: `1010001-0001`
- `1010001`: كود موظف الكول سنتر
- `0001`: رقم العميل التسلسلي

### لوحات تحكم متخصصة

#### لوحة موظف الكول سنتر
- عرض الراتب اليومي والحوافز في الوقت الفعلي
- تسجيل عملاء جدد وإعطاء أكواد
- متابعة إحصائيات الأداء
- قائمة العملاء مع حالة الحضور

#### لوحة موظف الريسبشن
- البحث عن العملاء بالكود
- تسجيل حضور العملاء
- تسجيل الأجهزة للصيانة
- عرض الحوافز اليومية

#### صفحة تتبع العميل
- البحث بكود العميل
- عرض جميع الأجهزة المسجلة
- متابعة حالة الصيانة
- تاريخ كامل لخطوات الإصلاح

## التقنيات المستخدمة

- **Frontend**: Next.js 15, React 19, TypeScript
- **Backend**: Supabase (PostgreSQL)
- **UI**: Tailwind CSS, shadcn/ui
- **Authentication**: Supabase Auth
- **Real-time**: Supabase Realtime

## البدء السريع

### المتطلبات
- Node.js 20+
- pnpm
- حساب Supabase

### التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/egyptofrance/modeer.git
cd modeer

# تثبيت المكتبات
pnpm install

# إعداد متغيرات البيئة
cp .env.local.example .env.local
# قم بتعديل .env.local وإضافة بيانات Supabase

# تطبيق migrations على قاعدة البيانات
supabase db push

# تشغيل المشروع
pnpm dev
```

### الوصول للواجهات

- لوحة الكول سنتر: `http://localhost:3000/employee/call-center`
- لوحة الريسبشن: `http://localhost:3000/employee/reception`
- تتبع العميل: `http://localhost:3000/customer-tracking`

## قاعدة البيانات

تم إنشاء 10 جداول رئيسية:

1. **employee_types** - أنواع الموظفين (7 أنواع)
2. **employees** - بيانات الموظفين
3. **customers** - بيانات العملاء
4. **devices** - الأجهزة المسجلة
5. **incentives** - الحوافز
6. **attendance** - الحضور والانصراف
7. **work_schedules** - مواعيد العمل
8. **incentive_settings** - إعدادات الحوافز
9. **device_status_history** - تاريخ حالة الأجهزة
10. **holidays** - الإجازات الرسمية

## الأمان

تم تطبيق Row Level Security (RLS) على جميع الجداول لضمان:
- كل موظف يرى بياناته فقط
- المدراء يرون جميع البيانات
- العملاء يرون أجهزتهم فقط
